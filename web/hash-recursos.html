Estandares y especificaciones

FIPS

FIPS 180-4 — Secure Hash Standard (SHA-2)

Especificacion oficial de SHA-224, SHA-256, SHA-384, SHA-512 y variantes. Documento de referencia NIST para SHA-2.

csrc.nist.gov →
FIPS

FIPS 202 — SHA-3 Standard

Especificacion de SHA3-224, SHA3-256, SHA3-384, SHA3-512 y SHAKE128/SHAKE256 basados en la permutacion Keccak.

csrc.nist.gov →
RFC

RFC 2104 — HMAC

Definicion de HMAC (Hash-based Message Authentication Code). La construccion ipad/opad que se usa en JWT, TLS y APIs REST.

datatracker.ietf.org →
RFC

RFC 5869 — HKDF (HMAC-based Key Derivation Function)

Especifica HKDF, basado en HMAC. Usado en TLS 1.3 para derivacion de claves de trafico (handshake, application).

datatracker.ietf.org →
NIST SP

NIST SP 800-131A Rev.2 — Transitioning Cryptographic Algorithms

Define que MD5 y SHA-1 estan prohibidos para funciones de seguridad. Referencia para auditar configuraciones.

csrc.nist.gov →

Implementaciones de referencia

Python

hashlib — stdlib Python

Libreria estandar con MD5, SHA-1, SHA-256, SHA-512, SHA3-256, BLAKE2. Usa OpenSSL internamente. Siempre preferir sobre implementaciones manuales.

docs.python.org →
Web API

Web Crypto API — SubtleCrypto.digest()

API nativa del navegador para SHA-1, SHA-256, SHA-384, SHA-512. Sin dependencias externas. No incluye SHA-3 ni MD5.

MDN →
C / OpenSSL

OpenSSL EVP_DigestInit / SHA256()

Implementacion de produccion para hashing en C/C++. Incluye todas las variantes SHA-2, SHA-3, BLAKE2 y HMAC.

openssl.org →
Keccak

Keccak Team — Reference Implementation

Implementacion de referencia oficial del equipo Keccak/SHA-3. Incluye codigo C optimizado y vectores de prueba.

keccak.team →

Herramientas SOC y DFIR

Threat Intel

VirusTotal — busqueda por hash

Permite buscar muestras de malware por MD5, SHA-1 y SHA-256. Herramienta fundamental para correlacion de IOCs en SOC L1/L2.

virustotal.com →
DFIR

sha256sum / certutil / Get-FileHash

Herramientas nativas para calcular hashes en Linux/macOS (sha256sum), Windows CMD (certutil -hashfile) y PowerShell (Get-FileHash).

PowerShell docs →
DFIR

Autopsy / FTK — integridad de evidencia

Las suites forenses calculan automaticamente SHA-256 de cada evidencia adquirida para garantizar la cadena de custodia.

autopsy.com →
EDR / SIEM

Sysmon — Event ID 1 (Process Create)

Sysmon registra el hash SHA-256 de cada proceso creado. Permite correlacion directa con IOCs de plataformas de threat intelligence.

learn.microsoft.com →
Password

Hashcat — password auditing

Herramienta de auditoria de passwords. Permite demostrar la debilidad de hashes planos (MD5, SHA-1, SHA-256 sin salt) frente a ataques de diccionario y rainbow tables.

hashcat.net →

Lectura aplicada

Paper

SHAttered — SHA-1 Collision (Google / CWI, 2017)

Demostracion practica de la primera colision SHA-1. Dos PDFs con contenido diferente y el mismo SHA-1. Incluye codigo de verificacion y vectores.

shattered.io →
NIST

NIST Hash Function Competition — SHA-3 Selection Report

Informe oficial sobre el proceso de seleccion de SHA-3. Explica los criterios de seguridad, rendimiento y diseño que llevaron a elegir Keccak.

csrc.nist.gov →
SOC

Hunting con hashes — MITRE ATT&CK

Tecnicas de deteccion y hunting basadas en hashes de artefactos maliciosos. T1027 (Obfuscation) y T1036 (Masquerading) incluyen ejemplos de correlacion por hash.

attack.mitre.org →
